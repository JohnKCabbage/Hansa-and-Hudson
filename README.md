## Overview

**Hansa & Hudson** is an independent analytical practice focused on transforming real-world data into **actionable strategic insight**.

This repository serves as a portfolio of:

- Reproducible analytical workflows  
- OSINT-driven research  
- Quantitative strategic assessment  
- Interactive dashboards  
- Written intelligence-style products  

All projects are built using **publicly available data and transparent methodology**.

- Convert raw data into decision-ready analysis  
- Apply technical tools to real geopolitical and financial problems  
- Demonstrate end-to-end analytical capability  
- Produce operationally relevant research
